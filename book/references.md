# References & Further Reading

## Books

| Title | Author(s) | Why Read It |
|-------|-----------|-------------|
| **Python Crash Course, 3rd Edition** | Eric Matthes | The best hands-on introduction for beginners |
| **Automate the Boring Stuff with Python, 2nd Edition** | Al Sweigart | Learn Python by building useful real-world scripts |
| **Fluent Python, 2nd Edition** | Luciano Ramalho | Deep dive into Python's idioms and internals |
| **Beyond the Basic Stuff with Python** | Al Sweigart | What comes after the beginner stage |
| **Python Tricks** | Dan Bader | A buffet of awesome Python features |
| **Effective Python, 2nd Edition** | Brett Slatkin | 90 specific ways to write better Python |
| **Learning Python, 5th Edition** | Mark Lutz | A comprehensive (if lengthy) reference |

:::{note}
Most of these books are available online for free through library services like [O'Reilly Online Learning](https://learning.oreilly.com/) if you have access.
:::

## Documentation

- [Python Standard Library](https://docs.python.org/3/library/) --- Every built-in module documented
- [Python Language Reference](https://docs.python.org/3/reference/) --- The formal language specification
- [PEP 8 -- Style Guide](https://peps.python.org/pep-0008/) --- How to write idiomatic, readable Python
- [PEP 257 -- Docstring Conventions](https://peps.python.org/pep-0257/) --- How to document your code
- [The Python Wiki](https://wiki.python.org/moin/) --- Community-maintained knowledge base

:::{important}
Following PEP 8 from the start will make your code easier to read and share with others. Many teams enforce it with tools like `flake8` or `ruff`.
:::

## Cheat Sheets

- [Python 3 Memento](https://perso.limsi.fr/pointal/python:memento) --- A dense one-page reference (PDF)
- [Comprehensive Python Cheat Sheet](https://gto76.github.io/python-cheatsheet/) --- Community-maintained, exhaustive
- [Beginners Python Cheat Sheet (pynative)](https://pynative.com/python-basics-cheat-sheet/) --- Concise beginner reference
- [Python Crash Course Cheat Sheets](https://ehmatthes.github.io/pcc_2e/cheat_sheets/) --- By the author of *Python Crash Course*
- [Official Python Download](https://www.python.org/downloads/) --- Get the latest Python

:::{note}
Cheat sheets are great for quick reference, but don't rely on them too heavily while learning. Make sure you understand *why* something works, not just *what* to type.
:::
