# How to Use This Book

---

## Reading Online

The book is hosted as a fully interactive website. You can read it directly in your browser at the book's URL. All content is rendered using the [Jupyter Book](https://jupyterbook.org/) platform, which means code examples are live and executable.

## Downloading PDF

:::{tip}
You can download a PDF version of the entire book by clicking the **Download** button in the top toolbar and selecting **PDF**. This is perfect for offline reading or printing.
:::

## Using Jupyter Notebooks

Every chapter is backed by a Jupyter Notebook. You can download any notebook and run it locally:

1. Click the **download** icon (⭳) at the top of any page
2. Choose **`.ipynb`** to download the notebook file
3. Open it in VS Code, JupyterLab, or Jupyter Notebook
4. Run cells interactively and experiment with the code

:::{warning}
Make sure you have Jupyter installed. See the [setup](setup) guide for help setting up your environment.
:::

## Interactive Features

### Copy Buttons

Every code block has a **copy** button in the top-right corner. Click it to copy the code to your clipboard.

### Code Tabs

Some examples show multiple approaches or languages. Click between tabs to view different versions of the same concept.

### Dark Mode

Toggle dark mode using the 🌙/☀️ icon in the top toolbar. This reduces eye strain during long study sessions.

### Admonitions

Throughout the book you will find special callout boxes:

:::{note}
**Notes** provide additional context or information that complements the main text.
:::

:::{tip}
**Tips** offer shortcuts, best practices, and helpful suggestions.
:::

:::{warning}
**Warnings** alert you to common pitfalls and mistakes to avoid.
:::

:::{important}
**Important** boxes highlight critical concepts you must understand before moving on.
:::

## Progress Tracking

There is no built-in progress tracker, but here are some suggestions:

- Bookmark the page you are currently studying
- Keep a study journal noting which days you complete
- Use the checkboxes in the table of contents to track chapters

## Getting Help

If you get stuck:

1. Re-read the lesson and try the examples step by step
2. Check the [troubleshooting](#troubleshooting) section in the setup guide
3. Search or ask on [GitHub Discussions](https://github.com/your-org/python-book/discussions)
4. Open an issue on [GitHub Issues](https://github.com/your-org/python-book/issues)

:::{important}
Please **do** ask questions! There are no stupid questions --- only unanswered ones. The community is here to help.
:::
