## What You'll Learn

1. **Python Fundamentals** — Variables, data types, and basic I/O
2. **Control Flow** — Conditionals, loops, and logical operators
3. **Functions** — Writing reusable code with parameters and return values
4. **Data Structures** — Lists, tuples, dictionaries, and sets
5. **File I/O** — Reading from and writing to files
6. **Object-Oriented Programming** — Classes, inheritance, and more
7. **Beyond Basics** — Decorators, generators, and working with libraries
8. **Capstone Project** — Build a complete real-world application

## How This Book Is Structured

This book is organized into **7 weekly modules**, designed to be completed over 30--40 days:

| Week | Topic | Duration |
|------|-------|----------|
| 1 | Python Fundamentals | Days 1--6 |
| 2 | Control Flow | Days 7--12 |
| 3 | Functions | Days 13--18 |
| 4 | Data Structures | Days 19--24 |
| 5 | File I/O | Days 25--30 |
| 6 | Object-Oriented Programming | Days 31--36 |
| 7 | Beyond Basics & Capstone | Days 37--40 |

Each day includes a short lesson, hands-on exercises, and a checkpoint quiz. Weekly mini-projects reinforce your learning.

---

## Features

- **📖 Interactive Code Cells** — Run Python code directly in your browser
- **✏️ Hands-On Exercises** — Practice each concept as you learn it
- **✅ Checkpoint Quizzes** — Test your understanding at the end of each day
- **🔨 Weekly Mini-Projects** — Apply what you've learned to real problems
- **🏆 Capstone Project** — A final project that ties everything together
- **🌙 Dark Mode** — Comfortable reading day or night
- **📱 Mobile-Friendly** — Learn on any device
